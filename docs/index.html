Introduction

Eat the food(it will appear with heart shape) Don't crush on the wall!

The Tutorial

1. If you want to start, press any key.
2. You can move your snake with keys(w:up, s:down, a:left, d:right).
3. Eat the food(the heart shape), and your snake's tail will be longer.
4. If you crush on your own snake body or the wall, you fail. So the game will be overed.

-- If you want to pause the game while playing, push the key 'p'.
-- You can see the current score, the best score and the last game score.
